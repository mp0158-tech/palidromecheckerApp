Use Case 1 – Application Entry & Welcome Message in a pointwise format:

Objective
To display the application name, version, and welcome message when the program starts.

-->>Execution Flow
Step 1: The JVM invokes the main() method.

Step 2: The Application name is displayed.

Step 3: The Application version is displayed.

Step 4: A startup confirmation message is shown.

Step 5: The program either exits or moves to the next use case.

-->>Core Concepts Used
Class: The fundamental building block of the Java application.

=Main Method: The entry point for program execution.

=Static Keyword: Used for methods that belong to the class rather than an instance.

=Console Output: Utilizing System.out.println to print text to the terminal.

=Application Flow Control: Managing the sequence of operations within the program.

-->>Expected Output
=A formatted header showing "Palindrome Checker App".

=Detailed lines for the Use Case title and Version: 1.0.

=A final status message: "Application Started Successfully!".------------------------------------------
